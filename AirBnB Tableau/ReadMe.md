#  Real Estate Price & Revenue Dashboard

An interactive dashboard analyzing property prices and revenue trends by **year, zip code, and bedroom count**.

---

##  Overview

This dashboard provides insights into:

-  **Price Per Year (2016)** – Weekly trend of property prices throughout the year.
-  **Price by Zip Code** – Comparison of average prices across multiple zip codes.
-  **Price Per Zip Code (Map View)** – Geographical distribution of property prices.
-  **Revenue for Year by Bedrooms** – Average price based on bedroom count.
-  **Bedroom Filter** – Interactive filter to refine analysis by number of bedrooms.

---

##  Key Insights

- Property prices show an overall upward trend during 2016.
- Noticeable variation exists between different zip codes.
- Properties with more bedrooms generate higher average revenue.
- Geographic visualization highlights premium and mid-range areas.

---

##  Tools Used

- Tableau (Data Visualization & Dashboard Development)
- Mapbox / OpenStreetMap (Geospatial Mapping)

---

##  Purpose

This dashboard supports:

- Real estate market analysis  
- Investment decision-making  
- Pricing strategy evaluation  
- Location-based performance comparison
